Зачем шлюз — и почему этот#

Один ключ — много моделей

GPT, Claude, Gemini, DeepSeek, Kimi и другие за одним endpoint и на одном счёте. Без отдельных регистраций, карт и сверок.

Три протокола нативно

OpenAI Chat Completions, OpenAI Responses и Anthropic Messages — Claude Code работает как надо, а не через обёртку.

Честность по замыслу

Без тихой подмены. Список моделей на уровне аккаунта, прозрачный расход и способ поймать даунгрейд.

Проверяемо, а не на веру

Направьте открытый бенчмарк на DaoXE и на кого угодно. Прогон воспроизводимый — перезапустите сами.

Встраивается в ваши инструменты

Меняете один base URL в Cursor, Cline, Aider, LobeChat и SDK OpenAI/Anthropic. Существующий код продолжает работать.

Для зарубежных разработчиков

Для разработчиков за рубежом и русскоязычных разработчиков (СНГ и релоканты). Недоступно в материковом Китае.

Старт за три шага#

  1. Получите ключ. Зарегистрируйтесь и создайте API-ключ в личном кабинете DaoXE.
  2. Узнайте ID моделей. ID зависят от аккаунта — вызовите endpoint моделей и скопируйте точный ID. Не берите строку из чужого блога.
  3. Направьте клиент на base URL. Base URL https://daoxe.com/v1, ключ и точный ID модели. Для Claude Code — нативный путь Messages.
bash
export DAOXE_API_KEY="ваш_ключ"

curl --fail-with-body --show-error --silent \
  https://daoxe.com/v1/models \
  -H "Authorization: Bearer ${DAOXE_API_KEY}"

Не верьте на слово — сравните бенчмарком#

«Быстро и надёжно» — это утверждение; число, которое можно перезапустить, — это доказательство. Проект llm-gateway-benchmark измеряет связность, p50/p95 задержку и цену за 1M токенов. Он не хранит ваш ключ, промпт и ответы. Как запустить — в гайде по бенчмарку.

Проверяемость — весь смысл

Направьте CLI на DaoXE, на официальный API и на текущего провайдера, затем сравните. Если провайдер уходит от вопроса про бенчмарк — это и есть ответ. Узнайте, как распознать подмену модели.

Оплата и реальная стоимость#

Alipay / WeChat Pay / USDT — фиксированно 1 ¥ = $1

Любой способ пополняет по фиксированному курсу 1 ¥ = $1, а модели тарифицируются по своей официальной цене в USD. При реальном курсе ~7 ¥/$ западные флагманы (GPT / Claude / Gemini / Grok) выходят примерно в 1/7 официальной цены (≈на 85% дешевле) — и через USDT тот же курс получают держатели USD за рубежом. Честная оговорка: дешёвые китайские модели (DeepSeek / Qwen / GLM) тоже выгоднее прямого доступа, но меньше — это не «всё в 1/7». Курсы меняются; daoxe.com/pricing — источник истины, а небольшое пополнение это подтвердит.

Частые вопросы#

DaoXE OpenAI-совместим?

Да. Base URL https://daoxe.com/v1 и ваш ключ; существующий код на OpenAI SDK работает без изменений. Также доступны OpenAI Responses и нативный Anthropic Messages.

Claude Code работает через DaoXE?

Да — нативно по протоколу Anthropic Messages на /v1/messages, поэтому вызов инструментов и стриминг работают корректно. См. гайд по Cursor и Claude Code.

Какой ID модели указывать?

Всегда точный ID из GET /v1/models для вашего аккаунта — ID зависят от аккаунта, не копируйте чужие строки.

Как убедиться, что модель настоящая?

Запустите открытый воспроизводимый бенчмарк против DaoXE и официального API и сравните. См. как распознать подмену модели.

Есть ли бесплатный старт?

Стартовый бонус: уточняется. Актуальные условия — на официальном сайте.
