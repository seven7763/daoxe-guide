Доступ к Claude API — OpenAI-совместимо и нативно для Claude Code

Два пути к Claude через DaoXE: OpenAI-совместимый chat и нативный Anthropic Messages, на котором работают Claude Code и SDK Anthropic.

Обновлено 2026-07-20

Claude работает через DaoXE либо с OpenAI SDK (просто смените base URL), либо через нативный /v1/messages. Messages — для Claude Code и SDK Anthropic; chat — для OpenAI-клиентов. Цены на странице цен — текущее преимущество см. страницу цен. Где реально экономия — гайд по дешёвым API.

Как вызвать Claude через DaoXE#

OpenAI-совместимый путь — /v1/chat/completions с ключом Bearer. Нативный путь — /v1/messages с заголовками x-api-key и anthropic-version. Для Claude Code укажите ANTHROPIC_BASE_URL на корень хоста — путь он добавит сам.

bash
curl --fail-with-body --show-error --silent \
  https://daoxe.com/v1/messages \
  -H "x-api-key: ${DAOXE_API_KEY}" \
  -H "anthropic-version: 2023-06-01" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "YOUR_EXACT_MODEL_ID",
    "max_tokens": 64,
    "messages": [{"role": "user", "content": "Reply with OK."}]
  }'
bash
# Claude Code reads the host ROOT; it appends /v1/messages itself
export ANTHROPIC_BASE_URL="https://daoxe.com"
export ANTHROPIC_AUTH_TOKEN="your_api_key"   # some builds read ANTHROPIC_API_KEY
# optional: pin exact catalog IDs from GET /v1/models
export ANTHROPIC_MODEL="paste_exact_model_id_here"
export ANTHROPIC_SMALL_FAST_MODEL="paste_small_model_id_here"

Нюансы именно для Claude#

  • Claude Code использует КОРЕНЬ хоста (https://daoxe.com), не .../v1 — он сам добавит /v1/messages. См. гайд Cursor и Claude Code.
  • Заголовки нативного Messages иные: x-api-key + anthropic-version, а не Bearer (это путь chat).
  • Extended thinking — опционально. Включайте в запросе, когда нужны видимые рассуждения; иначе обычный ответ.
  • Берите точные ID из GET /v1/models — не вписывайте строку версии из блога.

Убедитесь, что отдают настоящую модель#

Сначала проверьте эндпоинт, а не клиент — если это не работает, настройки не помогут:

bash
export DAOXE_API_KEY="your_api_key"

# List the exact model IDs your account can call
curl --fail-with-body --show-error --silent \
  https://daoxe.com/v1/models \
  -H "Authorization: Bearer ${DAOXE_API_KEY}"

Частые вопросы#

Можно Claude с OpenAI SDK?

Да — base URL https://daoxe.com/v1 и /v1/chat/completions с ID модели Claude.

Claude Code работает через DaoXE?

Да, нативно через /v1/messages. Задайте ANTHROPIC_BASE_URL=https://daoxe.com (корень). Подробности в гайд Cursor и Claude Code.

x-api-key или Bearer?

Нативный Messages — x-api-key + anthropic-version; OpenAI-совместимый chat — Bearer.

Сколько стоит Claude здесь?

По модели и аккаунту — см. актуальные цены. Текущее преимущество к официалу: см. страницу цен.
