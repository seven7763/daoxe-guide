Gemini API 的 OpenAI 兼容接入 —— 一个 key，无需单独 Google 密钥

Google 为 Gemini 提供了 OpenAI 兼容端点；通过 DaoXE 你完全无需单独的 Google AI Studio 密钥，直接用 Gemini 模型 ID 调 /v1/chat/completions。

更新于 2026-07-20

无需自己管理 Google AI Studio 密钥和它的 base URL，DaoXE 把 Gemini 放在与 GPT、Claude、DeepSeek 同一个 key 后面。用 gemini-* ID 调 /v1/chat/completions。分流见最便宜 API 教程。

通过 DaoXE 调用 Gemini#

用 /v1/chat/completions，模型填从 GET /v1/models 取的准确 Gemini ID。在支持视觉的模型上，多模态图片输入走 OpenAI 风格的 content 数组。

bash
curl --fail-with-body --show-error --silent \
  https://daoxe.com/v1/chat/completions \
  -H "Authorization: Bearer ${DAOXE_API_KEY}" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "YOUR_EXACT_MODEL_ID",
    "max_tokens": 64,
    "messages": [{"role": "user", "content": "Say hello in one sentence."}]
  }'

Gemini 特有的注意点#

  • 无需单独的 Google 密钥。 通过 DaoXE，Gemini 用你的同一个 DaoXE key —— 不必注册 Google AI Studio。
  • 多模态走 content 数组。 在支持视觉的 ID 上，用 OpenAI 风格的消息 content 数组发图片。
  • 参数映射有差异。 少数 OpenAI 参数在 Gemini 上行为不同；请求尽量简单并测试。
  • 只用准确 ID。 从 GET /v1/models 读取 gemini-*。

支付与真实成本#

支付宝 / 微信 / USDT —— 统一 1 元 = $1

所有支付方式统一按 1 元 = $1 额度充值，模型按各自的美元官方价扣费。按真实汇率约 7，西方旗舰（GPT / Claude / Gemini / Grok）实付约为官方美元价的 1/7（省约 85%）——USDT 同样按此充值，海外持美元者也享此价。诚实提醒：国产廉价模型（DeepSeek / Qwen / GLM）也比官方直连便宜，只是幅度小些，并非「所有模型都 1/7」。汇率会变，实时价以定价页为准，充一小笔即可自证。

常见问题#

需要 Google AI Studio 密钥吗？

不需要 —— 通过 DaoXE，Gemini 在你的同一个 DaoXE key 后面；无需单独注册 Google。

怎么调用 Gemini？

用 /v1/chat/completions，填从 GET /v1/models 取的准确 gemini-* ID。

能发图片吗？

支持视觉的 ID 可以 —— 用消息里的 OpenAI 风格 content 数组。

这里 Gemini 多少钱？

按模型、按账号 —— 见实时定价。相对官方优势：多数主流模型经低价分组约低 30%–80%。
