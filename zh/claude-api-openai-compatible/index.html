Claude API 接入 —— OpenAI 兼容，且为 Claude Code 原生支持

通过 DaoXE 调 Claude 有两条路：OpenAI 兼容的 chat，以及 Claude Code 和 Anthropic SDK 使用的原生 Anthropic Messages。

更新于 2026-07-20

Claude 通过 DaoXE 有两种用法：用 OpenAI SDK（只改 base URL），或走原生 /v1/messages。Claude Code 和 Anthropic SDK 用 Messages；OpenAI 形态的客户端用 chat。价格见定价页 —— 当前优势 多数主流模型经低价分组约低 30%–80%。省钱到底靠什么，见最便宜 API 教程。

通过 DaoXE 调用 Claude#

OpenAI 兼容路径是 /v1/chat/completions，用 Bearer key。原生路径是 /v1/messages，用 x-api-key 和 anthropic-version 头。Claude Code 的 ANTHROPIC_BASE_URL 要指向主机根 —— 路径它会自己补。

bash
curl --fail-with-body --show-error --silent \
  https://daoxe.com/v1/messages \
  -H "x-api-key: ${DAOXE_API_KEY}" \
  -H "anthropic-version: 2023-06-01" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "YOUR_EXACT_MODEL_ID",
    "max_tokens": 64,
    "messages": [{"role": "user", "content": "Reply with OK."}]
  }'
bash
# Claude Code reads the host ROOT; it appends /v1/messages itself
export ANTHROPIC_BASE_URL="https://daoxe.com"
export ANTHROPIC_AUTH_TOKEN="your_api_key"   # some builds read ANTHROPIC_API_KEY
# optional: pin exact catalog IDs from GET /v1/models
export ANTHROPIC_MODEL="paste_exact_model_id_here"
export ANTHROPIC_SMALL_FAST_MODEL="paste_small_model_id_here"

claude

Claude 特有的注意点#

  • Claude Code 用主机根（https://daoxe.com），不是 .../v1 —— 它会自己补 /v1/messages。见Cursor·Claude Code 教程。
  • 原生 Messages 的头不同： 用 x-api-key + anthropic-version，不是 Bearer（那是 chat 路径）。
  • 扩展思考是可选的。 需要显式推理时在请求里开启；否则返回普通答案。
  • 用准确 ID（来自 GET /v1/models）—— 别照搬博客里的版本字符串。

支付与真实成本#

支付宝 / 微信 / USDT —— 统一 1 元 = $1

所有支付方式统一按 1 元 = $1 额度充值，模型按各自的美元官方价扣费。按真实汇率约 7，西方旗舰（GPT / Claude / Gemini / Grok）实付约为官方美元价的 1/7（省约 85%）——USDT 同样按此充值，海外持美元者也享此价。诚实提醒：国产廉价模型（DeepSeek / Qwen / GLM）也比官方直连便宜，只是幅度小些，并非「所有模型都 1/7」。汇率会变，实时价以定价页为准，充一小笔即可自证。

常见问题#

能用 OpenAI SDK 调 Claude 吗？

能 —— base URL 设为 https://daoxe.com/v1，用 Claude 模型 ID 调 /v1/chat/completions。

Claude Code 能走 DaoXE 吗？

能，原生走 /v1/messages。设 ANTHROPIC_BASE_URL=https://daoxe.com（主机根）。完整步骤见Cursor·Claude Code 教程。

用 x-api-key 还是 Bearer？

原生 Messages 用 x-api-key + anthropic-version；OpenAI 兼容 chat 用 Bearer。

这里 Claude 多少钱？

按模型、按账号 —— 见实时定价。相对官方的当前优势：多数主流模型经低价分组约低 30%–80%。
